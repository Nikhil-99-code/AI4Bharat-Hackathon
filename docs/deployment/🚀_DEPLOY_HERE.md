## 🆘 Need Help?

### Quick Fixes:

**App won't start?**
```bash
pip install -r requirements.txt --force-reinstall
```

**Port 8501 busy?**
```bash
# Windows
netstat -ano | findstr :8501
taskkill /PID <pid> /F
```

**AWS errors?**
- Check `.env` file exists
- Verify credentials are correct
- Ensure region is `ap-south-1`
